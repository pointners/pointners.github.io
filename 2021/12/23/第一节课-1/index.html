接下来我们开始讲解每行代码的含义
//代表的是注释，通常用来注解代码的意义

其中我们来解释下endl
我们可以尝试运行下列程序，就知道加不加endl的区别了

#include <iostream>
using namespace std;

int main()
{
    cout << "hello";
    cout << "world";
    return 0;
}

这是另一个 

#include <iostream>
using namespace std;

int main()
{
    cout << "hello" << endl;
    cout << "world" << endl;
    return 0;
}

比较一下就知道endl是什么意思了
接下来讲一下如何赋值一个变量
基本语法：变量类型 + 变量名 = 变量初始值
这里又牵扯到变量类型

int 表示这个变量是一个整数

float 表示这个变量是一个浮点数（小数）

大部分情况都可以用int，如果有特殊情况后面会再讲

  1. 常见的错误：
    • 符号必须是全英文的，不能有中文
    • cout前面的空格叫做缩进，通常按tab键，一个缩进=四个空格
